Этот сайт — результат одного вечера и серии промптов в Claude Code. Вот как он устроен.

## Стек

- **Astro 6** — статический генератор, быстрый и гибкий
- **Tailwind CSS v4** — utility-first CSS через `@tailwindcss/vite`
- **Content Collections** — типизированный контент на Markdown/MDX
- **Astro View Transitions** — плавные переходы между страницами
- **GitHub Pages** — деплой через GitHub Actions

## Архитектура

Сайт построен вокруг двух коллекций контента:

**Блог** — записи в Markdown с frontmatter: заголовок, описание, дата, теги, привязка к проекту.

**Проекты** — 13 направлений, каждое со своей цветовой точкой, описанием и связанными постами.

Посты привязываются к проектам через поле `project` или теги. Страница проекта автоматически подтягивает связанные записи.

## Дизайн

Минималистичный. Монохромная палитра — чёрный текст на светлом фоне, белый на тёмном. Акценты — только через цветные точки проектов. Шрифт Inter Variable. Noise-текстура для глубины.

Никаких эмодзи-иконок, градиентов, теней на кнопках. Типографика как главный инструмент визуальной иерархии.

## Тёмная тема

Inline-скрипт в `<head>` читает `localStorage` до загрузки CSS — никакого мерцания при переключении. Tailwind v4 через `@custom-variant dark`.

## SEO

- `<html lang="ru">` и `og:locale="ru_RU"` для русскоязычного поиска
- JSON-LD structured data на каждой странице
- Sitemap, robots.txt с директивой для Яндекса
- Title всегда начинается с «Хирахмо» — работает на узнаваемость бренда

## Процесс создания

Всё было сделано через Claude Code — от `npm create astro` до `git push`. Я описывал что хочу, ИИ писал код. Несколько итераций на дизайн, одна на SEO, одна на соцсети.

Общее время: около 2 часов. Из них активной работы (формулирование промптов, ревью) — может минут 30.

Это не магия. Это новая нормальность.
