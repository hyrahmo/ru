В рамках [365-дневного челленджа](/blog/365-days-of-ai/) я использую разные AI-инструменты. Вот текущий стек.

## Основные

**Claude (Anthropic)** — основной ассистент. Подписка Max ($100/мес). Использую для:
- Программирования через Claude Code
- Анализа и исследований
- Написания текстов
- Работы с базой знаний

**Claude Code** — CLI-инструмент от Anthropic для работы с кодом прямо в терминале. Этот сайт создан через него. Умеет читать файлы, писать код, запускать команды, деплоить.

## Для кода

- **Claude Code** — основной инструмент для разработки
- **GitHub Copilot** — автодополнение в VSCode
- **Cursor** — пробовал, но вернулся к Claude Code

## Для контента

- **Claude** — генерация и редактура текстов
- **Midjourney / DALL-E** — изображения (когда нужны)

## Для автоматизации

- **AI-агенты** — кастомные агенты на базе Claude Agent SDK
- **MCP-серверы** — подключение внешних инструментов к Claude

## Чего не использую

- Не использую ИИ для финальных решений — только как инструмент
- Не использую для генерации бессмысленного контента ради объёма
- Не использую для обмана — всегда указываю, где помог ИИ

## Критерии выбора

Инструмент должен:
1. Реально ускорять работу, а не создавать иллюзию продуктивности
2. Давать контроль — я должен понимать и проверять результат
3. Интегрироваться в рабочий процесс, а не требовать отдельного «окна»

Список будет обновляться по мере появления новых инструментов.
